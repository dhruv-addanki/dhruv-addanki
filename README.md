## 🚀 Current focus

**Scanis:** AI fitness built around 3D scans, body data, and adaptive training.

✧ SwiftUI iOS app with Firebase-backed auth, data, and storage flows<br>
✧ Scan-to-insight flow from body capture to personalized workout plans<br>
✧ Product loop across user interviews, waitlist, mobile testing, and shipping

**SuperAgent:** WhatsApp-based AI assistant for daily workflows.

✧ Connects Gmail, Google Calendar, Google Drive, and Asana behind deterministic backend tools<br>
✧ Supports voice input, daily briefings, approval-gated writes, audit logs, and reply-aware task actions<br>
✧ Built as a production-minded Node/TypeScript backend with Prisma, Redis, queues, and OAuth integrations

**Personal Context Graph:** Obsidian plugin that turns ChatGPT exports into readable memory.

✧ Imports conversation history into typed Markdown nodes for projects, people, preferences, decisions, and artifacts<br>
✧ Merges carefully so the graph gets more connected instead of noisier<br>
✧ Generates an agent-ready context pack while keeping notes readable in a normal vault
